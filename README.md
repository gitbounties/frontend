## Running for development

Install dev tools
```sh
npm i -g prettier
```

Set up git hooks
```sh
npm run devsetup
```

Create local copy of environment variables
```sh
cp .env-example .env.local
```

Install dependencies
```sh
npm i
```

First, run the development server:
```sh
npm run dev
```
